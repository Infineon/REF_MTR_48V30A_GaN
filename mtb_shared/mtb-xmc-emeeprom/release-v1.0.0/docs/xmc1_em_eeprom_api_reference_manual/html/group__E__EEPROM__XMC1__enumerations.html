enum E_EEPROM_XMC1_STATUS_t

Defines the error codes for E_EEPROM_XMC1 APP.

Enumerator
E_EEPROM_XMC1_STATUS_SUCCESS 

Successful completion of APP initialization.

E_EEPROM_XMC1_STATUS_FAILURE 

Initialization failed due to internal operational error.

E_EEPROM_XMC1_STATUS_UNINITIALIZED 

APP is in Uninitialized state.

E_EEPROM_XMC1_STATUS_IDLE 

APP is in IDLE state.

Only used in E_EEPROM_XMC1_GetStatus() API

E_EEPROM_XMC1_STATUS_BUSY 

APP is in BUSY state.

Only used in E_EEPROM_XMC1_GetStatus() API

enum E_EEPROM_XMC1_OPERATION_STATUS_t

Defines current status for E_EEPROM_XMC1 APP.

Enumerator
E_EEPROM_XMC1_OPERATION_STATUS_SUCCESS 

Successful completion of operation.

E_EEPROM_XMC1_OPERATION_STATUS_FAILURE 

APP failed due to internal flash operational errors.

E_EEPROM_XMC1_OPERATION_STATUS_INCONSISTENT_BLOCK 

Targeted block is inconsistent.

E_EEPROM_XMC1_OPERATION_STATUS_INVALID_BLOCK 

The latest copy of the targeted block was invalidated.

E_EEPROM_XMC1_OPERATION_STATUS_CRC_FAILED 

CRC verification failed for the targeted block.

E_EEPROM_XMC1_OPERATION_STATUS_NOT_ALLOWED 

Operation not allowed to continue.

E_EEPROM_XMC1_OPERATION_STATUS_MEMORY_BANK_FULL 

Write/Invalidate operation failed when bank has no space.

Call E_EEPROM_XMC1_StartGarbageCollection API to proceed

E_EEPROM_XMC1_OPERATION_STATUS_NO_CRC_WRITTEN 

Written block does not have CRC checksum enabled.
